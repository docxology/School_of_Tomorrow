## Metadata Standards

### Required Fields
```yaml
---
title: Document Title
type: document_type
tags: [tag1, tag2]
created: YYYY-MM-DD
updated: YYYY-MM-DD
status: [draft|active|archived]
---
```

### Optional Fields
```yaml
related: [[Doc1]], [[Doc2]]
contributors: [[User1]], [[User2]]
version: 1.0.0
aliases: [Alt1, Alt2]
```
